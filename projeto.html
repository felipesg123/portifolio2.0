Projetos

Aqui estão alguns dos meus projetos:

Projeto 1: Sistema de Login

Um sistema de login moderno e responsivo com opções para redefinir senha e registrar um novo usuário.

Projeto 2: Todo Avançado

Um aplicativo de lista de tarefas avançado com funcionalidades de adicionar, pesquisar e filtrar tarefas.

Projeto 3: Cardápio Interativo

Um cardápio digital interativo com seções para pratos principais, bebidas, sobremesas e tabelas de bolos.

Projeto 4: Calendário de Veículos

Um calendário interativo para gerenciar a entrada e saída de veículos, com diferentes categorias e restrições de uso.
